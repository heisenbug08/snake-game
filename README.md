# Snake Game

Simple snake game made with [Raylib](https://github.com/raysan5/raylib)

### Build Instructions
```bash
git clone --depth=1 https://github.com/heisenbug08/snake-game.git
cd snake-game
cmake -B build -G Ninja && cd build
ninja

# Run
./snake
```
